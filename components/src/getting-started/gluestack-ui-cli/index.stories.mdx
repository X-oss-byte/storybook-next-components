# Command Line Interface (CLI) Guide

Gluestack-ui is a command line tool that helps to streamline your development experience. This CLI provides commands for adding, updating, or removing components in your project.
`gluestack-ui` CLI helps to setup gluestack-ui in your project.

- It adds `gluestack-ui-.config.ts` file in your project root directory. This file contains all the configuration and theme that will be used in components of gluestack-ui.
- It also adds `GluestackUIProvider` and `styled` wrapper in the path provided by you while setting up the app.

It also provides a list of commands which make managing of the components easier for you. You can add, update, or remove components using these commands.

```bash

# init and add components
npx gluestack-ui@latest

# add component
npx gluestack-ui@latest add <component-name>

# add all components
npx gluestack-ui@latest add --all

# update component
npx gluestack-ui@latest update <component-name>

# update all component
npx gluestack-ui@latest update --all

# remove component
npx gluestack-ui@latest remove <component-name>

# remove all component
npx gluestack-ui@latest remove --all

# help
npx gluestack-ui@latest help
```
